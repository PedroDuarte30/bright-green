# 🌿 Plant Shop - Landing Page

A modern, responsive, and performance-optimized landing page designed for a boutique plant shop. This project follows a **Mobile-First** approach, ensuring a seamless user experience across all devices, from smartphones to large desktop screens.



## 🚀 Technologies Used

* **HTML5:** Semantic structure for better accessibility and SEO.
* **CSS3:** Advanced styling using CSS Variables, Flexbox, and CSS Grid.
* **JavaScript (Vanilla):** Logic for the hamburger menu, dynamic scroll-triggered header, and interface interactions.
* **Mobile-First Design:** Optimized CSS architecture for performance and scalability.

## ✨ Features

* **Smart Navigation:** Header that transitions styles upon scrolling and a fully functional interactive mobile menu.
* **Impactful Hero Section:** Cinematic overlays and fluid typography using `clamp()` for perfect scaling.
* **Product Catalog:** Responsive grid layout with hover effects and "New" product badges.
* **Fully Responsive:** Adapts gracefully to mobile, tablet, and desktop viewports.
